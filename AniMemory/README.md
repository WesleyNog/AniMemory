# Image Memory Game – One Piece Theme

Este projeto é um jogo da memória com imagens, desenvolvido em SwiftUI como estudo e prática dos conceitos modernos de desenvolvimento para Apple platforms. As cartas do jogo trazem imagens inspiradas na série One Piece.

## 🎮 Objetivo
Recriar um clássico jogo da memória onde o jogador precisa encontrar pares de imagens iguais. A interface é responsiva, animada e utiliza recursos modernos do SwiftUI.

## 🚀 Técnicas e Conceitos Utilizados
- **SwiftUI**: Construção de interface declarativa e reativa
- **MVVM**: Separação clara entre lógica de negócios (ViewModel) e interface (View)
- **@ObservedObject**: Comunicação entre ViewModel e View para atualização automática da UI
- **LazyVGrid**: Grid responsivo para exibir cartas dinamicamente
- **Animações & Transições**: Uso de animações suaves ao virar cartas, revelar pares, e reiniciar o jogo
- **Componentização**: Criação de views reutilizáveis, como a CardView
- **Bindings & State**: Controle de estado local e global para atualização da interface
